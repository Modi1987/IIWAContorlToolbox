The folder Matlab_client (this folder), contains the scripts (Matlab files) of the ICT (IIWA Control Toolbox) Toolbox.

Breakdown of contents (for this folder):
1- The main file of the Toolbox is the ICT.m class, which contains the various methods to control the robot.
2- Example scripts (tutorials) showing how to use the ICT.m class are also included. Those example tutorials start with the initials ICTclass_Tutorial_***.m. Check those tutorials to test the toolbox and for a rapid hands-on experience.
